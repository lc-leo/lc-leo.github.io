如何在Hexo上发布第一篇博客

发表于 2018-01-04 | 分类于 Hexo | |

1、 新建md格式文档

在本地hexo 文件夹下右键点击Git Bash Here

默认在 ~/source/_posts 文件夹下会新建名为“The-Blog-Title.md”的markdown文件。

2、 编辑你的博文

推荐马克飞象——一款界面漂亮、可同步预览的markdown编辑器。

3、 文章预览

执行hexo server服务，在浏览器中打开 http://localhost:4000/ 就可以预览了。

4、 把文章做静态化处理

在本地 ~/public 目录下生成对应的html、JavaScript、css文件。

5、 发布到GitHub

在hexo主页上就会看到新发布的文章了。
