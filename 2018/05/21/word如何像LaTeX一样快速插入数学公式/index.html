word如何像LaTeX一样快速插入数学公式

发表于 2018-05-21 | 分类于 经验分享 | |

前言

如果文章中含有大量的数学符号和公式，那无疑用LaTeX可以实现最完美最高效的写作，但是LaTeX学习成本太高，有大量内容需要记忆。有人可能会推荐大名鼎鼎的数学公式编辑软件MathType，但它是收费的，只能免费适用30天。word自带的公式编辑器基本上就可以满足所有的公式插入要求，但用鼠标点击来输入公式效率太低，而且如果想插入一个不常用的符号需要找很长时间。
如果不懂LaTeX语法，如何快速用键盘输入的方法插入公式呢？在word中可以用UnicodeMath编辑的方式快速插入公式

UnicodeMath的原理

我们遇到的所有符号都被编入了Unicode码中，而我们输入的所有内容都是这些Unicode编码的集合。所以当插入符号公式时，只需调出来相应的Unicode码即可。
跟程序语言中的转义字符一样，用“\”加特定的语句将其转化为Unicode编码中的相应字符。公式的输入也以“\”开始，以空格结束。

插入公式

Alt + = ：插入一个公式输入框
Ctrl + I ：将字体变为斜体

自定义公式

  1. 如果觉得默认的命令不好记或不符合你的习惯，可以自定义符号或公式的命令。
    比如我想使用命令’\b’然后空格就插入β,只需要打开【公式选项】-【数学符号自动更正】，将\beta替换为\b即可。
  2. 还可以自定义公式内容
    比如我想定义一个常用公式，当我输入\hi时插入hello/leo，只需要在【数学符号自动更正】中在【替换】一栏输入“\hi”，在【替换为】一栏输入“hello/leo”即可

    注

  3. 上标或下标在前面时：^229_90空格Th空格
    $_{90}^{229}Th$
  4. 大部分符号其实都可以借助强大的搜狗输入法来输入。（格式可能不符合要求）
    比如要输入符号α，只需打出alpha搜狗就给你列出这个符号了。或者你想输入积分符号∫，只需输入jf，搜狗就给你找出来了。。。
