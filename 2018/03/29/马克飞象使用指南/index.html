马克飞象使用指南

发表于 2018-03-29 | 分类于 学习笔记 | |

插入目录

正文开头输入[TOC]

标题

  1. 第一种标题
    H1：#… or Ctrl + 1
    H2： ##… or Ctrl + 2
    …….
  2. 第二种标题
    H1：标题下加二个以上=
    H2：标题下加二个以上-

字体

  1. 斜体：星号 or …
    Ctrl + I

  2. 粗体：两个星号 or …
    Ctrl + B

  3. 字体背景-深色标签@(…)

  4. 字体背景-浅色标签@[…|…]

插入链接

Ctrl + L

  1. 行内式链接 [name]（address）

  2. 参考式链接 [描述][标记]

插入图片

Ctrl + G
or ！[@图片描述|图片位置|图片尺寸]（图片地址）
图片描述会显示在图片下方；
图片位置：right，left，center
图片尺寸：e.g. 400*600像素，0表示自适应尺寸

还可插入附件，网络图片等

插入引用

句首加 >

——维基百科

代码编写

Ctrl + K

  1. 行内代码(单撇)

gROOT->Reset();

  1. 整行代码（三撇）
